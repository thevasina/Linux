1. Управление пользователями:
	- sudo useradd -s /bin/bash -d /home/user1 -m janekrol
	- sudo userdel janekrol
	- sudo adduser alex

2. Управление группами:
	- sudo groupadd family
	- sudo addgroup coworkers
	- sudo usermod -g family alex
	- sudo usermod -aG coworkers alex
	- sudo gpasswd -d alex coworkers

3. Добавить пользователя, имеющего право выполнять команды/действия от имени суперпользователя.  Сделать так, чтобы sudo не требовал пароль для выполнения команд.
	- sudo adduser janekrol
	- sudo usermod -aG sudo janekrol
	- sudo visudo
	- janekrol ALL=(ALL) NOPASSWD:ALL - добавляется в конце файла

4. Используя дополнительные материалы, выдать одному из созданных пользователей право на выполнение ряда команд, требующих прав суперпользователя (команды выбираем на своё усмотрение). 
	- с помощью команды which найти точное расположение (я выбрала команды apt и reboot)
	- sudo visudo
	- thevasina ALL=(ALL) NOPASSWD:/usr/sbin/reboot /usr/bin/apt
